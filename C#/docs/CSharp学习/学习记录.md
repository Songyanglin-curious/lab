## 2026-07-01

知识点：01. 认识 CSharp 语言
是否通过：是，基础通过
关键结论：C# 是语言，.NET 是平台，ASP.NET Core 是框架；C# 源码通常编译为 IL 和元数据，再由 .NET 运行时执行；框架能力是基于语言和平台的高层封装；SDK 是开发期工具箱，Runtime 是运行期环境，TargetFramework 是目标平台声明，LangVersion 是语言规则版本。
主要误区：C# 不等同于语言标准；.NET 不只是运行环境；IL 不是 CPU 直接执行的机器码；C# 支持 AOT 不代表常规模型等同于 C/C++；.NET 版本不等同于 C# 语言版本；SDK 不等同于 Runtime；NuGet 包兼容性不是简单看包版本号大小。
后续问题：后续再深入 CLR、JIT、AOT、程序集、元数据、SDK 构建流程、MSBuild、NuGet 资产选择等细节。
