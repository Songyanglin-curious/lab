# CSharp 学习说明

这里保存 C# 系统学习资料、总清单、知识点目录和后续沉淀内容。

学习方式：一次一个点，讲清、问透、练透、沉淀，通过检查后再进入下一个点。

当前主线只聚焦 C# 语言本身，不提前展开 ASP.NET Core、EF Core、部署等框架和工程主题。

## 目录说明

- `学习清单.md`：总学习进度，完成一个点就勾选一个点。
- `官方资料.md`：官方资料入口和使用规则。
- `学习记录.md`：记录每次学习的日期、知识点和结论。
- `知识点/`：每个知识点一个目录，用于沉淀说明、对话、总结和检查题。

## 学习原则

- 先理解概念，再写代码。
- 先掌握主线，再补边角。
- 先说清边界，再进入框架。
- 不用“我大概知道”作为通过标准。
