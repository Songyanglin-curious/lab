# 官方资料

本学习系统以官方资料为主，其他公开资料只作为辅助解释。

## 核心资料

- Microsoft Learn：C# 文档
- Microsoft Learn：C# 语言参考
- Microsoft Learn：C# 编程指南
- Microsoft Learn：.NET API 浏览器
- C# 语言规范

## 使用规则

- 优先看中文官方文档。
- 遇到翻译不清楚、术语不统一、示例不够明确时，对照英文官方文档。
- 涉及语法精确定义时，以 C# 语言规范为准。
- 非官方博客、视频、问答只能用来辅助理解，不能作为最终依据。
